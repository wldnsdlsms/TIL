# 재귀함수

날짜: 2024.07.26

## 재귀함수

 : 자기 자신을 호출하는 함수, 반복문과 동일함

 : 스택프레임 자료구조를 사용한다

 * 스택(Stack) 프레임 : 박스 쌓기(아래에서 위로 쌓기), 선입후출 또는 후입선출 구조라고 한다(LIFO - Last In First Out)

<예시>

```c
#include<studi.h>
#define MAX 4        //상수 MAX 선언

int main(){
	comp(MAX);
	return 0;
}
int comp(int n){
	if(n > 1)
		comp(n - 1);     //재귀함수 사용됨
	printf("%d", n);
} 

출력값 : 1 2 3 4
```
